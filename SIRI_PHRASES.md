# Siri Integration - Voice Commands

## How to Use Siri with Recall

After installing the app and logging in, you can use Siri to quickly save memories without opening the app.

### Basic Memory Commands

Say any of these phrases to Siri:

- "Remember [your thought] in Recall"
- "Save [your thought] to Recall"
- "Add memory [your thought] in Recall"
- "Store [your thought] in Recall"
- "Keep [your thought] in Recall"
- "Note [your thought] in Recall"
- "Record [your thought] in Recall"
- "Save thought [your thought] in Recall"
- "Remember to [your thought] in Recall"
- "Don't forget [your thought] in Recall"
- "Make a note [your thought] in Recall"
- "Add reminder [your thought] in Recall"
- "Store memory [your thought] in Recall"
- "Create memory [your thought] in Recall"
- "Add thought [your thought] in Recall"

**Example:**
- "Hey Siri, remember buy milk in Recall"
- "Hey Siri, save meeting notes in Recall"

### Urgent/High Priority Memories

For important memories that need immediate attention:

- "Remember urgently [your thought] in Recall"
- "Save urgent [your thought] in Recall"
- "Important [your thought] in Recall"
- "Urgent reminder [your thought] in Recall"
- "High priority [your thought] in Recall"

**Example:**
- "Hey Siri, remember urgently call doctor tomorrow in Recall"
- "Hey Siri, save urgent finish project deadline in Recall"

### Person-Related Memories

To save memories about specific people:

- "Remember [your thought] about [person name] in Recall"
- "Save [your thought] for [person name] in Recall"
- "Note about [person name] [your thought] in Recall"
- "Remember [person name] [your thought] in Recall"

**Example:**
- "Hey Siri, remember Mom's birthday is next week about Mom in Recall"
- "Hey Siri, save John prefers coffee for John in Recall"

## First Time Setup

1. Open Recall app and log in
2. Go to iPhone Settings > Siri & Search
3. Find "Recall" in the app list
4. Enable "Use with Siri"
5. Optional: Add shortcuts to Siri for quick access

## How It Works

- All Siri-created memories are automatically saved to Firebase
- Memories appear instantly in your Recall app
- They're marked with source: "siri" so you know they came from voice commands
- You can edit, complete, or delete Siri memories just like manual ones

## Privacy

- Your voice commands are processed by Apple's Siri
- Only the memory text is sent to Firebase
- All data is encrypted and secured
- You must be logged in for Siri commands to work
